## 훅 리스트

| 순서 | 훅                   | 실무 학습 포인트                                               |
| ------: | ---------------------- | ------------------------------------------------------------------------- |
|       1 | `useRef`               | DOM 접근, 이전 값 저장, 타이머/요청 id 저장          |
|       2 | `useReducer`           | 복잡한 상태 전이, action 기반 UI 상태 관리           |
|       3 | `useTransition`        | 무거운 렌더링을 낮은 우선순위로 처리              |
|       4 | `useActionState`       | 폼 submit, pending/error/success 상태 관리                        |
|       5 | `useId`                | 접근성 id, label-input 연결, SSR id 안정성                |
|       6 | `useDeferredValue`     | 검색/필터링 UI에서 입력 반응성 유지                |
|       7 | `useOptimistic`        | 좋아요/댓글/장바구니 같은 낙관적 UI 업데이트  |
|       8 | `useFormStatus`        | 폼 제출 중 pending 상태를 하위 컴포넌트에서 읽기 |
|       9 | `useSyncExternalStore` | 외부 store, browser API, custom subscription 연결                   |
|      10 | `useImperativeHandle`  | 부모에게 제한된 imperative API 노출                           |
