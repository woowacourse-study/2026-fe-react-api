## useRef

공식 문서의 표현

> `useRef`는 렌더링에 필요하지 않은 값을 참조할 수 있는 React Hook입니다.

```ts
const ref = useRef(initialValue);
```

`useRef` 훅을 사용해서 레퍼런스를 선언할 수 있다.

레퍼런스가 뭘까?

컴포넌트가 어떤 정보를 기억하고 싶지만, 그 정보의 변경으로 리렌더링을 유발하고 싶지 않을 때 사용하는 값이다.

더 자세히 말하면, ref는 문자열, 객체, 함수, DOM 노드 등 다양한 값을 가리킬 수 있다.  
state와 달리 ref는 읽고 수정할 수 있는 `current` 프로퍼티를 가진 일반 자바스크립트 객체이다.

한 마디로 정리하면, `useRef`는 렌더링에 직접 필요하지 않지만 컴포넌트가 기억해야 하는 값을 저장할 때 사용한다.

## useRef의 특징

- `useRef`는 `{ current: ... }` 형태의 객체를 반환한다.
- `ref.current` 값을 바꿔도 컴포넌트는 리렌더링되지 않는다.
- 컴포넌트가 다시 렌더링되어도 같은 ref 객체가 유지된다.
- 렌더링 결과에 영향을 주는 값은 ref가 아니라 state로 관리해야 한다.

ref는 React가 변경을 추적하지 않기 때문에, 화면에 보여야 하는 값을 ref에만 저장하면 UI가 업데이트되지 않는다.

`useRef`는 크게 2가지 역할을 수행한다.

1. 값을 참조하기
2. DOM 조작하기

### 1. 값을 참조하기

레퍼런스 값은 React가 추적하지 않는 컴포넌트의 비밀 주머니라고 볼 수 있다.

일반적으로 컴포넌트가 외부 API와 통신해야 할 때 ref를 사용할 수 있다. 특히 컴포넌트의 화면 형태에 영향을 미치지 않는 브라우저 API와 통신할 때 유의미하다.

예를 들면 다음과 같은 값을 저장할 때 사용할 수 있다.

- 타이머 id
- 이전 값
- DOM 노드
- 외부 라이브러리 인스턴스
- 요청 id

> 렌더링에 사용되지 않는 값을 저장하기 위한 `탈출구`이다.

### 2. DOM 조작하기

React는 렌더링 결과물에 맞춰 DOM 변경을 자동으로 처리하기 때문에 컴포넌트에서 직접 DOM을 조작할 일은 많지 않다.

그러나 특정 노드에 포커스를 옮기거나, 스크롤 위치를 옮기거나, 위치와 크기를 측정하기 위해 DOM에 접근해야 할 때가 있다.

#### 2-1. 포커스 이동

```tsx
import { useRef } from 'react';

function Form() {
  const inputRef = useRef<HTMLInputElement>(null);

  function handleClick() {
    inputRef.current?.focus();
  }

  return (
    <>
      <input ref={inputRef} />
      <button onClick={handleClick}>Focus the input</button>
    </>
  );
}
```

버튼을 누르면 input으로 포커스를 이동시킬 수 있다.

자세한 과정은 다음과 같다.

1. `inputRef`를 선언한다.
2. React가 `<input>` DOM 노드를 `inputRef.current`에 넣어준다.
3. `handleClick` 함수에서 `inputRef.current`의 DOM 노드를 읽는다.
4. 해당 DOM 노드의 `focus()` 메서드를 호출한다.

#### 2-2. 스크롤 이동

```tsx
import { useRef } from 'react';

function Cats() {
  const firstCatRef = useRef<HTMLImageElement>(null);
  const secondCatRef = useRef<HTMLImageElement>(null);

  function handleScrollToFirstCat() {
    firstCatRef.current?.scrollIntoView({
      behavior: 'smooth',
      block: 'nearest',
      inline: 'center',
    });
  }

  function handleScrollToSecondCat() {
    secondCatRef.current?.scrollIntoView({
      behavior: 'smooth',
      block: 'nearest',
      inline: 'center',
    });
  }

  return (
    <>
      <nav>
        <button onClick={handleScrollToFirstCat}>첫 번째 이미지</button>
        <button onClick={handleScrollToSecondCat}>두 번째 이미지</button>
      </nav>

      <ul>
        <li>
          <img
            src="cat_1.png"
            ref={firstCatRef}
            alt="첫 번째 고양이"
          />
        </li>
        <li>
          <img
            src="cat_2.png"
            ref={secondCatRef}
            alt="두 번째 고양이"
          />
        </li>
      </ul>
    </>
  );
}
```

`img`의 `ref`에 각각의 레퍼런스를 연결하면, 버튼 클릭 시 해당 이미지 위치로 스크롤을 이동시킬 수 있다.

## 정리

`useRef`는 값을 기억하지만 리렌더링을 발생시키지 않는다.

따라서 타이머 id, DOM 노드, 이전 값, 외부 API와 연결된 값처럼 화면 렌더링과 직접 관련 없는 값을 저장할 때 적합하다.

반대로 화면에 보여야 하거나 렌더링 결과를 바꿔야 하는 값은 `useState`를 사용해야 한다.
